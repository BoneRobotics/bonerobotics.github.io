[ 작업 규칙 ]
----------------------------------------------------------------
1. 한 번에 한 작업씩. 큰 변경은 단계적으로.
2. 영문을 수정하면 kr/ 한국어판도 반드시 같이 반영.
3. 수정한 HTML은 폴더 구조 그대로 (assets 상대경로 깨지 않게).
4. base64로 다시 합치지 말 것 — 분리 구조 유지 (편집용).
   (최종 배포용 단일 파일이 필요하면 그때 별도로 합치기 요청)
5. 작업 후 <div> 여닫기 균형 깨지지 않았는지 확인.
6. products.html은 비교적 무거우니 그 파일만 콕 집어 작업하면 좋음.
